# CodeAlpha_StudentGradeTracker
Student Grade Tracker – Project 

The Student Grade Tracker is a Java-based console application developed to efficiently manage and analyze student academic performance. The system allows users to add student names along with their grades, store the data using Java's ArrayList collection, and generate useful performance statistics.

The application provides core functionalities such as adding new student records, viewing all stored records, calculating the average score, identifying the highest and lowest marks, and displaying a complete summary report. It also supports  deleting student records, making the system more flexible and user-friendly.

The project follows an object-oriented programming (OOP) approach by separating the student data model from the application logic. Java classes, methods, loops, conditional statements, and exception handling are used to ensure efficient data management and smooth program execution. File I/O is implemented to save student records in a text file, allowing the data to persist even after the application is closed and reopened.

Overall, this project demonstrates practical implementation of Java fundamentals, including collections (ArrayList), OOP principles, file handling, user input validation, and statistical calculations, while providing a simple and interactive console-based interface for managing student grades.
